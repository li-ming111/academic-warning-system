## 使用说明

1. 访问登陆页面: `http://localhost:5179/login`
2. 输入对应用户端的账号和密码
3. 点击"登 录"按钮
4. 系统将自动跳转到对应角色的首页

## 功能模块

### 学生端功能
- 📊 学业看板 - GPA统计、预警概览
- 📈 成绩分析 - 成绩查看、趋势分析、申诉
- ⚠️ 预警管理 - 预警提醒、处理进度
- 🤝 帮扶计划 - 帮扶计划查看、进度追踪
- 📉 数据分析 - 详细统计、可视化图表
- ⚙️ 个人设置 - 密码修改、隐私设置、安全日志

### 教师端功能
- 📊 教学数据 - 学生统计、预警概览
- 📝 成绩管理 - 批量录入、成绩修改
- ⚠️ 预警处理 - 预警处理、沟通记录
- 📊 性能分析 - 成绩分布、学生表现
- 💬 反馈管理 - 学生反馈、回复反馈
- 📚 选修课管理 - 选课推荐、学分核查

### 辅导员端功能
- 👨‍🎓 学生概览 - 班级统计、预警趋势
- 👥 学生管理 - 学生列表、搜索、详情、通知
- ⚠️ 预警管理 - 预警列表、批量处理
- 📚 选修课管理 - 修读概览、学分核查

### 管理员端功能
- 📊 数据概览 - 全校统计、系统动态
- 🏫 学院管理 - 学院CRUD操作
- 📖 专业管理 - 专业编码、学号解析
- 👥 用户管理 - 用户列表、角色筛选、批量操作
- ⚙️ 规则管理 - 预警规则配置、规则测试
- 📊 数据统计 - 学生、预警、帮扶报表
- 📚 选修课中心 - 毕业要求、选课统计

## 技术实现

- **前端**: Vue 3 + Vite + Element Plus + ECharts
- **后端**: Spring Boot 3 + MyBatis-Plus + MySQL
- **身份认证**: JWT Token + localStorage
- **数据可视化**: ECharts 图表库

## 注意事项

- 演示账号使用本地存储实现，不需要真实的后端用户数据
- 登陆后 token 存储在 localStorage，刷新页面仍保持登陆状态
- 退出登陆会清除所有存储数据，返回登陆页面
