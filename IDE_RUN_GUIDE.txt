2. File > Open Project

4. 等待IDE索引完成（1-2分钟）

IntelliJ IDEA:
─────────────
1. 在主菜单选择 File > Project Structure
2. 确保 Project SDK 选择 Java 17 或以上
3. 点击项目中的 AcademicWarningApplication.java
4. 右键 > Run 'AcademicWarningApplication'
5. 或按 Shift+F10 快速运行

VS Code:
────────
1. 安装扩展:
   - Extension Pack for Java (Microsoft)
   - Spring Boot Dashboard (Pivotal)
2. 打开 src/main/java/com/academic/AcademicWarningApplication.java
3. 点击"Run"按钮或按 F5

  - 自动生成所有Getter/Setter方法
  - @Slf4j 正确注入log对象
