# 学业预警系统 - 整体架构说明

## 系统概述

学业预警系统是一个全栈Web应用，用于高校学业管理和预警。系统采用前后端分离架构，支持学生、教师、辅导员、管理员四个主要角色。

## 数据流设计

### 用户认证流程

```
┌─────────┐
│ 用户端  │
└────┬────┘
     │
     │ 1. 输入用户名/密码
     │
     ▼
┌──────────────────┐
│ Login.vue        │
│ (登录页面)       │
└────┬─────────────┘
     │
     │ 2. authAPI.login()
     │
     ▼
┌──────────────────────────────┐
│ AuthController.login()       │
│ POST /auth/login             │
└────┬─────────────────────────┘
     │
     │ 3. UserService.login()
     │
     ▼
┌──────────────────────────────┐
│ 验证用户名密码               │
│ 生成JWT Token                │
└────┬─────────────────────────┘
     │
     │ 4. 返回 LoginResponse
     │    { token, userId, role }
     │
     ▼
┌──────────────────────────────┐
│ Login.vue                    │
│ 保存 token 到 localStorage   │
└────┬─────────────────────────┘
     │
     │ 5. 跳转到对应角色首页
     │
     ▼
┌──────────────────────────────┐
│ Dashboard / 各功能页面       │
└──────────────────────────────┘
```

### API请求流程

```
┌─────────────────┐
│ Vue Component   │
└────┬────────────┘
     │
     │ import { studentAPI } from '@/api'

     │
     ▼

│ api/index.js            │

     │ return apiClient.get(`/students/dashboard/${userId}`)
     │

┌──────────────────────────────┐
│ api/client.js                │
│ (Axios配置)                  │
│ ├─ 请求拦截器: 添加token     │

└────┬─────────────────────────┘
     │
     │ GET /api/students/dashboard/{userId}
     │ Header: Authorization: Bearer {token}
     │
     ▼
┌────────────────────────────────┐
│ Spring Security Filter         │
│ (JWT Token验证)                │
└────┬───────────────────────────┘
     │
     │ ✓ Token验证成功
     │
     ▼
┌────────────────────────────────┐
│ StudentController              │
│ getDashboard(@PathVariable)    │
└────┬───────────────────────────┘
     │
     │ inject: StudentService
     │ inject: ScoreService
     │ inject: WarningService
     │
     ▼
┌────────────────────────────────┐
│ StudentService / ScoreService  │

│ (业务逻辑处理)                 │
└────┬───────────────────────────┘
     │
     │ StudentMapper.selectById(id)

     │
     ▼
┌────────────────────────────────┐
│ MyBatis-Plus Mapper            │
│ (ORM映射)                      │
└────┬───────────────────────────┘
     │
     │ SQL: SELECT * FROM users WHERE id = ?

     │
     ▼
┌────────────────────────────────┐
│ MySQL Database                 │
│ (数据库查询)                   │
└────┬───────────────────────────┘
     │
     │ 返回查询结果
     │
     ▼
┌────────────────────────────────┐
│ Mapper -> Service -> Controller│
│ (数据组装)                     │
│                                │
│ StudentDashboardResponse {     │
│   courseCount: 12,             │
│   gpa: 3.5,                    │
│   warningCount: 2              │
│ }                              │
└────┬───────────────────────────┘
     │
     │ ApiResponse.success(data)
     │ {
     │   "code": 200,
     │   "message": "success",
     │   "data": {...}
     │ }
     │
     ▼
┌────────────────────────────────┐
│ HTTP Response                  │
└────┬───────────────────────────┘
     │
     │ 响应拦截器: 检查code
     │ 如果 code === 200，解包返回 data
     │
     ▼
┌────────────────────────────────┐
│ Vue Component                  │
│ 接收数据并渲染UI               │
└────────────────────────────────┘
```

## 模块职责划分

### 后端模块

| 模块 | 职责 |
|-----|------|
| Controller | 接收HTTP请求，调用Service，返回响应 |
| Service | 实现业务逻辑，数据验证，事务处理 |
| Mapper | ORM映射，SQL执行 |
| Entity | 数据库实体映射 |
| DTO | 数据传输对象，API请求/响应 |
| Config | 应用配置，Spring Bean定义 |

### 前端模块

| 模块 | 职责 |
|------|------|
| Page/View | 页面展示，用户交互 |
| API | API方法定义，抽象后端调用 |
| Router | 路由配置，页面导航 |
| Store | 全局状态管理 |
| Client | HTTP客户端配置 |

## 数据库设计原则

### 表结构规范

1. **主键** - 统一使用 BIGINT AUTO_INCREMENT id
2. **时间戳** - created_at, updated_at（自动维护）
3. **索引** - 常查询字段建立索引
4. **外键** - 建立关联关系
5. **编码** - UTF-8mb4 支持Emoji

### 数据一致性

1. 使用事务确保数据一致性
2. 外键约束维护数据完整性
3. 唯一索引防止重复数据
4. 状态码规范化

## 安全设计

### 认证

- JWT Token认证
- Token自动过期（1小时）
- RefreshToken机制（可选）

### 授权

- 基于Role的权限控制
- 路由层面的权限检查
- API层面的权限验证

### 数据保护

- 密码加密存储（BCrypt）
- SQL注入防护（参数化查询）
- CORS跨域配置
- 敏感字段脱敏
