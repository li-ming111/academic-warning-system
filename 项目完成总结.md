## 运行和部署

### 本地开发运行

```bash
# 后端
cd 学业预警系统后端
mvn clean install
mvn spring-boot:run
# 访问 http://localhost:8080

# 前端
cd 学业预警系统前端
npm install
npm run dev
# 访问 http://localhost:5173
```

### 生产环境部署

```bash
# 后端构建JAR
cd 学业预警系统后端
mvn clean package -DskipTests
java -jar target/academic-warning-system-1.0.0.jar

# 前端构建静态文件
cd 学业预警系统前端
npm run build
# 将dist目录部署到Web服务器
```
